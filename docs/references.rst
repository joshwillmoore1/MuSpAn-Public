.. _references:
References
==========

This page provides references for the MuSpAn package and papers that have used MuSpAn.
Citing MuSpAn helps us to continue developing the package and supporting the community, so we appreciate your support.


Citing MuSpAn
-------------
If you use MuSpAn in your work, please reference the MuSpAn paper:
 

.. tab-set::

    .. tab-item:: Text
        :sync: text

        MuSpAn\: A Multiscale Spatial Analysis Toolbox. Joshua A. Bull, Joshua W. Moore, Shania M. Corry, Muyang Lin,  Hayley L. Belnoue-Davis,  Eoghan J. Mulholland, Simon J. Leedham and Helen M. Byrne. BioRxiv.  2024. DOI: https://doi.org/10.1101/2024.12.06.627195


    .. tab-item:: BibTeX
